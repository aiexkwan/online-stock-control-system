# 音效文件說明

請在此目錄下放置以下音效文件：

- `update.mp3` - 訂單更新時的提示音
- `success.mp3` - 訂單完成時的成功音效
- `error.mp3` - 錯誤發生時的警告音

## 推薦音效來源

1. **免費音效庫**
   - https://freesound.org/
   - https://www.zapsplat.com/
   - https://mixkit.co/free-sound-effects/

2. **建議音效類型**
   - update.mp3: 輕微的「叮」聲或通知音
   - success.mp3: 愉快的完成音效
   - error.mp3: 溫和的錯誤提示音

## 音效格式要求

- 格式：MP3
- 時長：0.5-2 秒
- 文件大小：< 100KB
- 音量：適中，避免過於刺耳
